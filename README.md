# ForFun

A tiny, fun project — first attempt! <3

---

## Overview

**ForFun** is a lightweight collection of small joke scripts. Just clone the repo and run the batch file — nothing fancy, just good vibes.

---

## Features

* Simple `.bat` script you can run instantly
* No setup required
* Perfect for quick fun or testing batch behavior
* Clean and tiny project structure

---

## How to Run

### Prerequisites

* Windows (since this uses a `.bat` script)
* Git (optional, for cloning the repository)

### Steps

1. Clone the repository:

```bash
git clone <your-repo-url>
```

2. Enter the jokes directory:

```bash
cd jokes
```

3. Run the script:

```bash
./forfun.bat
```
